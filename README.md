# 🐾 Adota Pet

Plataforma web para facilitar o processo de adoção de animais da ONG Love Patinhas.

## 📋 Sobre o projeto
Sistema que permite à ONG cadastrar animais disponíveis para adoção e 
a pessoas interessadas visualizar, filtrar e solicitar a adoção.
